# HTTP
1. 一次請求回應後，TCP關閉

## HTTP REQUEST MESSAGE
Mehtods: 有GET、POST兩種
User-Agent：使用端的身分

# HTTP RESPONSE MESSAGE
Code: 
* 200：OK
* 400：請求錯誤
* 404：網頁不存在
* 500：內部伺服器錯誤
* 503：

# Cookies
伺服器存在客戶端的小型記錄檔，可記錄登入站點之帳密以及卡號等，解決HTTP Stateless問題

# Session
伺服器端暫時存放資料的變數，如同PHP Session般，對於伺服器負擔較大，且連線關閉時就跟著消失

# FTP
檔案傳輸協定，PORT 21

# DNS
域名解析，PORT 53
本地端 DNS Server 會將查詢到的DNS 回應，記錄到記憶體內，稱DNS快取 (DNS Caching)
